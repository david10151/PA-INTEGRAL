CÓMO AGREGAR TUS IMÁGENES A NUTRIFORCE PERÚ
=============================================

El código de las 5 páginas ya está listo para mostrar tus fotos.
Solo tienes que crear la carpeta "img" (al mismo nivel que index.html,
css/ y js/) con esta estructura exacta, y subir cada archivo con el
NOMBRE EXACTO indicado abajo. En cuanto el archivo exista, reemplaza
automáticamente el placeholder punteado — no hay que tocar el HTML.

Si un archivo no existe todavía, la página sigue funcionando y
muestra el placeholder de siempre. No rompe nada.

RECOMENDACIONES
-----------------------
- Usa .jpg para fotos y .png si necesitas transparencia (si usas .png,
  recuerda cambiar la extensión en el HTML también).
- Respeta las dimensiones sugeridas o una proporción similar para que
  no se vean recortadas o deformadas (las imágenes se ajustan con
  object-fit: cover, que recorta los bordes sobrantes).
- Comprime tus imágenes antes de subirlas (por ejemplo con
  https://squoosh.app) para que el sitio cargue rápido.
- Sube esta carpeta "img" completa a tu hosting, en la misma carpeta
  donde están index.html, css/ y js/.

CÓMO FUNCIONA POR DENTRO (por si quieres tocarlo)
-----------------------
Cada placeholder ahora tiene, dentro del mismo bloque, una etiqueta
<img class="real-img" src="img/..." onerror="this.style.display='none'">.
Si el navegador no encuentra el archivo, se dispara "onerror" y la
imagen se oculta, dejando ver el placeholder punteado de siempre.
Si el archivo sí existe, la imagen se muestra encima y cubre el
placeholder por completo. No necesitas borrar nada del HTML.
